# governança-pericial

Framework conceitual para a prática do perito judicial.

---

## O que é este repositório

Este repositório documenta um conjunto de conceitos desenvolvidos a partir da prática real de perícia judicial — especificamente a construção do LabPeritus, sistema de gestão pericial, e da metodologia Praxis.

Não é documentação de produto. É um framework de pensamento sobre como peritos operam, decidem e sustentam o próprio trabalho técnico.

O material aqui é público e reutilizável. Se os conceitos forem úteis para pensar sobre prática pericial, gestão de trabalho técnico ou integridade de processo decisório, esse é o objetivo.

---

## Por que este framework existe

O perito judicial produz laudos. Mas o laudo é apenas a saída visível de um processo que raramente é documentado.

Entre a nomeação e a entrega existe uma sequência de decisões técnicas: qual método aplicar, quais evidências considerar, por que uma interpretação e não outra, onde houve incerteza e como ela foi resolvida. Esse percurso decisório não aparece no laudo. Existe apenas na memória de quem o tomou.

Quando o laudo é contestado, o perito precisa reconstruir o raciocínio. Quando um caso semelhante aparece meses depois, começa do zero. Quando a carga de processos cresce, a qualidade da decisão depende da energia disponível num dia específico.

O problema não é técnico. É estrutural.

Este framework nomeia esse problema e propõe conceitos para pensar sobre ele com mais precisão.
